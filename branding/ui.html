Component Library Recommendations
Navigation
Frosted Nav Bar
Fixed, backdrop-filter blur(28px), transparent bg with 1px border. Smooth color/opacity transition on scroll.
Motion
Custom Cursor System
Dual-layer: 8px filled dot + 36px trailing ring. mix-blend-mode: difference. Magnetic on interactive targets.
Hero
Animated Canvas Grid
requestAnimationFrame grid nodes with physics-based mouse repulsion. Architectural feel matches brand identity.
Scroll
IntersectionObserver Reveals
Staggered translateY + opacity reveals. 0.7s ease transitions. Delay multiplied by item index for cascade effect.
Projects
Asymmetric Mosaic Grid
CSS grid-template with span rows. Hover scale (1.05) on background layer only. Text layers remain stable.
Map
SVG Presence Map
Inline SVG with animated rotating dashes on active city dots. Connection lines with stroke-dasharray pulses.
Process
Horizontal Timeline
5-step flow with active/done state differentiation. Animated fill line driven by IntersectionObserver on scroll.
Cards
Leadership Portrait Cards
Gradient portrait with avatar ring pulse animation. Hover reveals LinkedIn link with stagger opacity transition.
Glass
Frosted Glass Variant
backdrop-filter blur(20px) saturate(1.3). Used for overlays, modals, and floating info panels on dark surfaces.
